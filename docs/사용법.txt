====================================================
🧹 안전하게 파일 지우기 (secure_delete.exe) - 사용법
====================================================

이 프로그램은 고른 파일을 무작위 데이터로 7번 덮어쓴 뒤 지워서,
일반 복구 프로그램으로는 되살릴 수 없게 만들어 줍니다.

★ 파이썬 등 어떤 프로그램도 설치할 필요가 없습니다.
   secure_delete.exe 파일 하나만 있으면 됩니다.

----------------------------------------------------
■ 실행하기 (아주 쉬움)
----------------------------------------------------
1) secure_delete.exe 를 더블클릭합니다.
   - 처음에 파란색 "Windows의 PC 보호" 창이 뜨면
     [추가 정보] → [실행] 을 누르세요.
     (인터넷에서 받은 새 프로그램이라 나오는 안내이며, 안전합니다.)

2) 창이 뜨면 순서대로:
     ① [📁 파일 고르기] 를 눌러 지울 파일을 고릅니다.
        ▶ 여러 개를 한 번에 고르려면 파일 선택 창에서
           Ctrl 을 누른 채 클릭하거나, Shift 로 범위를 선택하세요.
        ▶ [🗂 폴더 통째로] 를 누르면 그 폴더 안 모든 파일이 대상이 됩니다.

     ② 목록을 확인한 뒤 아래 빨간 [🗑 지우기] 버튼을 누릅니다.

     ③ "정말 지울까요?" 창에서 [예] 를 누르면 완전삭제됩니다.

----------------------------------------------------
■ 꼭 기억하세요! (안전 주의)
----------------------------------------------------
  ⚠ 한 번 지운 파일은 절대 되살릴 수 없어요.
     지우기 전에 목록의 파일 이름을 꼭 다시 확인하세요.

  ⚠ USB 메모리, SD카드, SSD 같은 저장장치는 기술적 특성(웨어 레벨링)
     때문에 완전삭제가 100% 보장되지 않을 수 있습니다.
     아주 중요한 비밀 파일이라면 저장장치 전체 암호화나
     물리적 파기도 함께 고려하세요.

----------------------------------------------------
■ 참고 (기술 정보)
----------------------------------------------------
  - 이 exe는 64비트(x64) Windows용입니다.
    Windows 11 ARM 노트북에서도 자동 호환 실행됩니다.
  - 동작 방식: 파일을 난수로 7회 덮어쓰기 → 크기 0으로 만들기
    → 무작위 이름으로 바꾸기 → 최종 삭제.
  - 함께 드린 secure_delete.py 는 같은 기능의 파이썬 버전입니다
    (Mac 등에서 쓰고 싶을 때 사용).
